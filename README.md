# Scholarship Portal

A front-end scholarship portal built with **HTML, CSS, and JavaScript** for student registration, login, scholarship application submission, application status tracking, and admin-side application review using browser `localStorage`.

## Overview

This project is a browser-based scholarship management prototype designed to simulate a simple student scholarship workflow. It includes separate student and admin interfaces, multi-step registration, scholarship application submission, status tracking, and an admin review dashboard.

The portal is intended as a **front-end academic/project prototype** and demonstrates how a complete multi-page workflow can be created using only core web technologies without external frameworks.

## Features

### Student Side
- Landing page with student and admin access
- Student sign-up with two-step registration flow
- OTP-style verification simulation
- Username and password creation
- Student login
- Scholarship application form submission
- Duplicate application prevention
- Application confirmation page
- Application status tracking

### Admin Side
- Admin login
- Admin dashboard
- View submitted applications
- Update application status
- Open student status view
- Review stored application details

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Browser LocalStorage
